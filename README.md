# db_hack

Позволяет внести изменения в электронный дневник ученика.

## fix_marks

Исправляет все двойки и тройки в электронном дневнике на пятерки.

Запускается командой:

```
fix_marks()
```

По умолчанию будут исправлены оценки Вани Фролова, для исправления оценок другого ученика - вписать в скобки его или её Фамилию Имя Отчество и заключить в кавычки, как в примере ниже:

```
fix_marks("Баранова Евфросиния Эльдаровна")
```

## remove_chastisements

Удаляет все замечания от учителей ученику.

Запускается командой:

```
remove_chastisements()
```

По умолчанию будут удалены замечания для Вани Фролова, для удаления замечаний для другого ученика - вписать в скобки его или её Фамилию Имя Отчество и заключить в кавычки, как в примере ниже:

```
remove_chastisements("Баранова Евфросиния Эльдаровна")
```

## create_commendation

В дневнике ученика появляется похвала от учителя, по случайному уроку с одним из хвалебных отзывов, указанного предмета.

Для работы этой функции, нужно указать название предмета, по которому требуется похвала, напрмер:

```
create_commendation("Математика")
```

или

```
create_commendation("Математика", "Баранова Евфросиния Эльдаровна")
```
если требуется похвала для другого ученика.

## Как запустить

Скачать скрипт в корневую папку сервера, где находится файл manage.py

В консоли перейти в корневую папку с сервером и запустить shell командой:

```
python manage.py shell
```

Импортировать вышеперечисленные функции командой:

```
from db_hack import fix_marks, remove_chastisements, create_commendation, get_schoolkid
```
Затем можно запускать функции, их аргументы и команды для запуска указаны в описании к функциям.
